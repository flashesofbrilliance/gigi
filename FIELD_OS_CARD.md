# ARCS v8.5 Field OS Card (for GIGI / ARCS-LOCAL)

## What This Is

- A **Field OS**, not a chatbot.
- A **self-clarification instrument**, not a generic decision assistant.
- A local-first substrate (GIGI) plus a decision OS (ARCS) that turns your judgment into an externalizable, queryable, inheritable asset.

## True North

> Each decision should increase the resolving power of your own judgment by making previously invisible dimensions of your decision geometry visible, queryable, and improvable.

## Operating Principles

- **NOTOMATION** – Tools may only externalize / scaffold / sequence cognition; they may not compress, replace, or overrule revealed preference.
- **Reference-of-frame** – The system externalizes your coordinate system so you can inspect the geometry of how you see, not just get more information about the world.
- **Local-first** – GIGI owns all state. Cloud LLMs, if used, are stateless compute only and never see raw session content, Decision Receipts, or transmission notes.
- **Triune Logic** – `NULL` (known absence), `OM` (running code), `UNDEFINED` (maximum-density frontier). Pure black is held, not clipped.
- **Heirloom** – The terminal asset is an honest, queryable map of how you saw the world — including blind spots — so someone who loves you can stand inside it later.

## Runtime Defaults

- **Front door:** Question Compass – “What are you really trying to decide?” with:
  - Horizon: today / this week / this quarter / 1–3 years.
  - Stakes: low / medium / high (or auto-inferred).
- **Modes:** QUICKMOVE / PLAN / COURT selected from stakes × reversibility.
- **Clarity modes:**
  - SIMPLE (default): 1 True North + ≤3 bullets of “why”.
  - RICH: SIMPLE + one short visual or extra explanation.
  - DEEP: full 12D court / advisor view, only on explicit request or clearly high stakes.
- **Drift Sentinel:** Every focused block starts with a JTBD. If we drift 3+ turns without artifacts, surface one land-the-plane prompt.

## Substrate Contract (ARCS ↔ GIGI)

- **GIGI stores:**
  - Templates as reusable facets of judgment (BAR / KKL / LAB / Decision Receipts / Drift / Kintsugi).
  - Gap logs as conscious OS eval signals (where the frame lacks resolution).
  - ROM welds as structural priors (patterns you no longer need to think about consciously).
  - Transmission notes and mistake cartography as heirloom metadata.
- **ARCS always queries GIGI first.**
- **Cloud** is a last resort, always explicit, and never sees raw life.

## Threat Model (Personal Use)

- In-scope and mitigated:
  - Cloud APIs reading decisions → avoided by local-first GIGI + structural-only L4/L5 (when used).
  - Disk failure / lost machine → mitigated via encrypted WAL checkpoints → iCloud with Advanced Data Protection.
  - Vendor data breach → iCloud holds ciphertext only.
- Out-of-scope (named and accepted):
  - State-level compulsion against Secure Enclave + ADP.
- Non-negotiable:
  - The wooden vats (ROM welds + receipts + notes) must never be destroyed.

Use this card for onboarding, README, and handoff. If someone reads nothing else, they should still understand what the system is, what it protects, and what it is for.
